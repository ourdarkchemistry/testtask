# User API

### Methods:

1. **POST** /api/users — Create a user
2. **PUT** /api/users/{id} — Update a user
3. **DELETE** /api/users/{id} — Delete a user
4. **POST** /api/login — Authentication
5. **GET** /api/users/{id} — Retrieve user information
